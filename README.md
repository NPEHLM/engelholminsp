Användarmanual: Tankekarta Program
1. Kom igång
När du startar programmet möts du av två val:
•	Starta ny tankekarta: Ger dig en tom arbetsyta med en central "Huvudidé". Du får först namnge din karta.
•	Ladda upp fil: Öppna en tidigare sparad tankekarta från din dator (en fil som slutar på .json).
2. Arbeta med noder (bubblor)
Noderna är byggstenarna i din tankekarta.
•	Skapa en nod: * Klicka på knappen + (Lägg till barn-nod) i verktygsfältet för att skapa en ny nod kopplad till den markerade noden.
o	Dubbelklicka var som helst på den rutmönstrade bakgrunden för att skapa en helt fristående nod.
•	Redigera text: Dubbelklicka på texten i en nod för att ändra den. Ett textredigeringsverktyg dyker upp där du kan göra texten fet, kursiv, ändra färg eller infoga en länk. Tryck på den gröna bocken för att spara.
•	Flytta: Klicka och dra en nod för att placera den var du vill på arbetsytan.
•	Rotera: Markera en nod. En liten blå cirkel visas ovanför noden – dra i den för att rotera noden.
•	Radera: Markera en nod och klicka på den röda soptunnan i verktygsfältet. (Obs: Du kan inte radera den sista noden).
3. Design och utseende
Du kan anpassa utseendet för att göra kartan mer överskådlig:
•	Former: Välj mellan rektangel, ellips, kvadrat, stjärna eller sol i verktygsfältet.
•	Färger: Använd den vänstra färgväljaren för att ändra färg på en markerad nod. Den högra färgväljaren ändrar bakgrundsfärgen på hela arbetsytan.
•	Pilar: Klicka på pil-ikonen för att visa eller dölja pilspetsar på kopplingslinjerna.
4. Bilder, Film och Dokument
Du kan göra dina noder mer informativa genom att bifoga media:
•	Bilder: Klicka på lila ikonen (landskap) för att ladda upp en bild direkt i noden.
•	Video: Klicka på gem-ikonen och välj "Ladda upp fil/film". Videon kan spelas upp direkt inuti noden.
•	Dokument: * Textfiler: Om du laddar upp en textfil (.txt, .md, .js) dyker knappen "Läs text" upp. Den låter dig läsa innehållet direkt i programmet.
o	Övriga filer: Andra filer (t.ex. PDF eller Zip) sparas som en nedladdningslänk i noden.
•	Webblänkar: Klicka på gem-ikonen och välj "Länk (URL)" för att koppla en nod till en extern webbsida.
5. Koppla ihop noder manuellt
Förutom de automatiska kopplingarna kan du skapa egna linjer:
1.	Klicka på kedje-ikonen i verktygsfältet (Koppla ihop noder manuellt).
2.	Klicka på den nod du vill starta ifrån.
3.	Klicka på den nod du vill koppla till. För att avbryta läget, klicka var som helst på bakgrunden.
6. Spara och Dela
Spara lokalt
Klicka på diskett-ikonen för att ladda ner hela tankekartan som en fil till din dator. Detta är det säkraste sättet att spara stora projekt med många bilder och filmer. Filen är i format .json och kan delas med andra.
7. Tips och Genvägar
•	Ångra/Gör om: Använd pilarna i verktygsfältet eller kortkommandona Ctrl+Z (ångra) och Ctrl+Y (gör om).
•	Panorera: Om kartan blir stor kan du scrolla i webbläsaren för att nå olika delar av arbetsytan.
•	Rensa allt: Knappen längst till höger i verktygsfältet tömmer hela kartan och låter dig börja om från början.
